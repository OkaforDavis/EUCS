## Official Names

### Full Legal Name
**Elugwu Umuoshie Farmer's Multipurpose Co-operative Society Ltd**

### Primary Name (Recommended)
**Elugwu Umuoshie Co-operative Society Ltd**

### Creative Abbreviations

#### 1. **EUCS** - Elugwu Umuoshie Co-operative Society
- **Use:** Official documents, letterheads, legal correspondence
- **Context:** "EUCS Member Portal", "EUCS Financial System"

#### 2. **EU Co-op** - Elugwu Umuoshie Cooperative
- **Use:** Marketing materials, member communications
- **Context:** "Welcome to EU Co-op", "EU Co-op Member Benefits"

#### 3. **Elugwu Co-op** - Short Form
- **Use:** Casual conversation, internal communications
- **Context:** "Elugwu Co-op Weekly Meeting"

#### 4. **EUMCS** - Elugwu Umuoshie Multipurpose Co-operative Society
- **Use:** When full designation is needed in abbreviated form
- **Context:** Bank statements, official registrations

#### 5. **The Society** - Informal Reference
- **Use:** Among members, internal documents
- **Context:** "The Society's annual general meeting"

## System Integration

### Code Prefix (Current System)
**`ect-`** prefix maintained for:
- CSS classes
- HTML IDs  
- JavaScript variables

*Rationale:* "ECT" loosely interpreted as "Elugwu Co-operative [Technology/Thrift]"
- Maintains code consistency
- No breaking changes needed
- Technical abbreviation separate from brand

### Database Tables
**`eucs_`** prefix recommended for new databases:
- `eucs_members`
- `eucs_contributions`
- `eucs_loans`
- `eucs_transactions`

## Do's and Don'ts

### ✅ DO:
- Use "Co-operative" with hyphen in formal contexts
- Capitalize properly: "Elugwu Umuoshie Co-operative Society Ltd"
- Include "Ltd" in legal documents
- Use EUCS abbreviation in technical/official contexts
- Maintain green color palette in all materials

### ❌ DON'T:
- Spell as "Cooperative" without hyphen in official docs
- Use inconsistent abbreviations (stick to approved list)
- Omit "Ltd" in legal/banking documents
- Use non-approved logo variations
- Stray from approved color palette
